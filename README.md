# 📚 Hands On - Estruturas de Controle II

## 📖 Descrição

Esta atividade tem como objetivo praticar conceitos fundamentais da linguagem Java, utilizando estruturas de controle, comandos de seleção, repetição, manipulação de strings e operações matemáticas.

Os exercícios propostos exploram diferentes situações que exigem raciocínio lógico e aplicação prática dos conteúdos estudados em sala de aula.

---

## 🎯 Objetivos

* Utilizar estruturas de seleção (`if`, `else` e `switch`).
* Aplicar estruturas de repetição (`do-while`).
* Manipular strings e caracteres.
* Realizar cálculos matemáticos utilizando a biblioteca `Math`.
* Desenvolver programas interativos com entrada de dados pelo usuário.
* Exercitar a lógica de programação em Java.

---

## 🛠️ Tecnologias Utilizadas

* Java
* JDK 17+ (ou versão utilizada na disciplina)
* Scanner para entrada de dados
* IDE de desenvolvimento (IntelliJ IDEA, Eclipse ou NetBeans)

---

## 📋 Exercícios Desenvolvidos

### 1. Impressão de Figura ASCII

Programa responsável por exibir no console a seguinte figura:

```text
 +"""""+
[| o o |]
 |  ^  |
 | '-' |
 +-----+
```

Conceitos aplicados:

* Saída de dados com `System.out.println()`
* Caracteres especiais em Strings

---

### 2. Distância entre Dois Pontos na Terra

Programa que calcula a distância entre duas coordenadas geográficas utilizando a fórmula:

```text
d = r * arccos(
sin(x1) * sin(x2) +
cos(x1) * cos(x2) * cos(y1 - y2)
)
```

Onde:

* r = 6371,01 km (raio da Terra)
* x = latitude
* y = longitude

Conceitos aplicados:

* Funções matemáticas da classe `Math`
* Conversão de graus para radianos
* Operações trigonométricas

---

### 3. Contador de Caracteres

Programa que recebe uma String e contabiliza:

* Quantidade de letras
* Quantidade de espaços
* Quantidade de números
* Quantidade de outros caracteres

Conceitos aplicados:

* Manipulação de Strings
* Estruturas de repetição
* Métodos da classe `Character`

---

### 4. Questão de Múltipla Escolha

Programa que apresenta uma pergunta relacionada a uma disciplina do semestre e utiliza:

* Estrutura `switch`
* Estrutura `do-while`

Regras:

* Exibir cinco alternativas (a, b, c, d e e).
* Permitir no máximo três tentativas.
* Informar quando a resposta estiver correta.
* Mostrar em qual tentativa o usuário acertou.
* Caso erre as três tentativas, exibir a mensagem:

```text
Resposta incorreta nas 3 tentativas
```

Conceitos aplicados:

* Estruturas condicionais
* Estruturas de repetição
* Controle de tentativas
* Interação com o usuário

---

## ▶️ Como Executar

1. Abra o projeto em uma IDE Java.
2. Compile o programa desejado.
3. Execute a classe correspondente ao exercício.
4. Informe os dados solicitados pelo sistema.
